How to Use

Use in GitHub Actions

Add your API key as a GitHub secret named FALSKY_API_KEY and reference it in your workflow.

YAML
- uses: Pritahi/falsky-action@v1
  with:
    api_key: ${{ secrets.FALSKY_API_KEY }}
    junit_path: 'test-results/*.xml'

Use in cURL

Pass your API key via the X-Falsky-API-Key header for write operations.

Shell
curl -X POST https://your-instance.com/api/junit \
  -H "X-Falsky-API-Key: YOUR_KEY_HERE" \
  -H "Content-Type: application/json" \
  -d '{"xml_content": "...", "repo_name": "my-repo"}'
